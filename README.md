# 📸 SnapCam App

SnapCam App is a Flutter-based mobile application that allows users to capture videos seamlessly using their device camera. The application provides a smooth video recording experience with modern UI and efficient performance.

---

## 🚀 Features

✅ Video capturing using device camera  
✅ Real-time camera preview  
✅ Simple and clean user interface  
✅ Fast video recording functionality  
✅ Cross-platform support (Android)  
✅ Built using Flutter framework  

---

## 🛠️ Tech Stack

- **Flutter**
- **Dart**
- **Android SDK**
- **Camera Plugin**
- **Material UI**
